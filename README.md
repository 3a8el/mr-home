# Mr. Home

> Your Vision in Wood, Meticulously Crafted.

A luxury interior design portfolio website for Mr. Home — a specialized manufacturer of custom Kitchens, Dressing Rooms, and Architectural Woodwork.

## Tech Stack

- **HTML / CSS / Vanilla JS** — no framework
- **GSAP + ScrollTrigger** — animations and scroll-driven effects
- **Lenis** — smooth scroll
- **Google Fonts** — Montserrat

## Project Structure

```
mr-home/
├── index.html          # Home page
├── pages/
│   ├── about.html      # About Us
│   ├── projects.html   # Projects grid
│   ├── contact.html    # Contact form
│   └── project.html    # Single project view
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # All scripts
└── assets/
    └── images/         # Project images
```
